Tecnologías

Backend
- **Node.js** + **Express.js**
- **MongoDB** con Mongoose
- **JWT** para autenticación
- **bcryptjs** para encriptación de contraseñas
- **Morgan** para logging

Frontend
- **React** con Vite
- **React Router DOM** para routing
- **Context API** para estado global
- **Lazy Loading** y **Suspense** para optimización
